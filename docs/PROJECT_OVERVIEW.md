# 肌骨康复速查 V5.0 专业版 - 项目概述

## 1. 项目简介

**肌骨康复速查**是一款面向康复医学专业人员的临床辅助工具，提供肌肉骨骼系统疾病的评估量表、康复治疗方案、临床指南等一站式查询服务。

### 1.1 核心功能

| 功能模块 | 描述 |
|---------|------|
| 评估量表 | 50+种临床评估量表（疼痛、关节功能、平衡、肌力等） |
| 临床工具 | 关节活动度、肌力分级、痉挛评估、步态分析 |
| 疾病知识库 | 25+种疾病的评定量表映射与临床指南 |
| 康复方案 | 循证康复治疗方案（物理治疗、作业治疗、言语治疗） |
| 疼痛管理 | 常见肌骨疼痛的分期治疗方案 |
| 肌肉速查 | 详细的肌肉功能、损伤、评估及治疗信息 |

### 1.2 技术特性

- **纯前端单页应用**：无服务器依赖，可离线运行
- **响应式设计**：适配手机、平板、桌面设备
- **模块化架构**：数据与逻辑分离，易于扩展
- **一键构建**：支持单文件打包发布

## 6. 核心设计原则

### 6.1 模块化数据结构

所有数据模块遵循统一的JSON结构，便于遍历和渲染：

```javascript
{
  id: '唯一标识符',
  name: '显示名称',
  category: '分类',
  description: '描述',
  content: { ... }
}
```

### 6.2 评估量表统一接口

评估量表均实现 `calculate()` 方法，统一评分逻辑：

```javascript
{
  id: 'vas',
  name: 'VAS 视觉模拟疼痛评分',
  calculate: function(answers) {
    return { score: answers[0] || 0, maxScore: 10 };
  }
}
```

### 6.3 康复方案分期结构

康复方案采用标准化分期格式：

```javascript
{
  id: 'pt-cmt',
  name: '颈椎手法治疗康复方案',
  stages: [
    { name: '急性期', goal: '目标', duration: '时长', exercises: [...] }
  ]
}
```
