# 医智学 - 肌骨康复学习软件（离线版）

## 📦 下载离线版本

**文件名**: `yizhixue-offline.html`
**大小**: 约 3 MB
**所有内容已内嵌到单个 HTML 文件中**

## 🚀 使用方法

### 方法 1：直接双击打开
1. 下载 `yizhixue-offline.html`
2. 双击文件，会自动用默认浏览器打开
3. 无需网络，无需服务器

### 方法 2：拖到浏览器
1. 下载文件
2. 打开 Chrome / Safari / Edge 等浏览器
3. 把文件拖到浏览器窗口中

### 方法 3：手机使用
1. 把文件传到手机（微信/网盘等）
2. 用手机浏览器打开
3. 添加到主屏幕，作为 PWA 使用
